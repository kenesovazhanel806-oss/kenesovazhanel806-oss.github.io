
# ✈️ TravelTour - Әуе билеттерін іздеу және брондау жүйесі

Бұл жоба **"Backend және дерекқорларға кіріспе"** пәні аясында ұйымдастырылған топтық жобалық жұмыс. Жүйе пайдаланушыларға отандық әуе компаниялары (Air Astana, FlyArystan, SCAT) арқылы билет іздеуге, багаж таңдауға және тиімді рейстерді брондауға мүмкіндік береді.

## 👥 Команда мүшелері және рөлдері
* **Жанель (Kenesovazhanel806)** — Фронтенд әзірлеу (HTML/CSS/JavaScript), жобаның кіріспе бөлімі және презентация, GitHub репозиторийін басқару.
* **Команда мүшесі 2** — Backend логикасы (API құру, Аутентификация, Валидация).
* **Команда мүшесі 3** — Дерекқор архитектурасы (PostgreSQL/MySQL), кестелерді құру және SQL сұраныстары.

---

## 📅 Жобаның күнделікті орындалу барысы (Есеп)

### 🔹 1-күн: Жобаны жоспарлау және команда құру (Орындалды)
* Команда құрылды және жобаның тақырыбы ретінде "TravelTour" авиациялық порталы таңдалды.
* GitHub репозиторийі іске қосылып, негізгі құрылымы жасалды.

### 🔹 2-күн: Ортаны баптау және Git негіздері (Орындалды)
* Жобаның негізгі фронтенд файлдары (`index.html`) репозиторийге сәтті жүктелді (Commit & Push).
* `.gitignore` файлы қосылды.

### 🔹 3-4 күн: Дерекқорды жобалау және SQL (Жоспарда)
* Дерекқордың ER-диаграммасын (Users, Flights, Bookings кестелері) құру.
* Дерекқорды 3-ші қалыпты формаға (3NF) келтіру.
